🚀 **Selenium Automation Framework**

📌 **Project Overview**

This project is a test automation framework built using Selenium WebDriver with Java, designed to automate web application testing efficiently and maintainably.

It follows industry best practices like **Page Object Model (POM)** and supports scalable test execution using **TestNG** and **Maven**.

🛠️ **Tech Stack**

* **Language:** Java
* **Automation Tool:** Selenium WebDriver
* **Testing Framework:** TestNG
* **Build Tool:** Maven
* **API Testing:** Postman (Basic), RestAssured (Intro level)
* **Version Control:** Git & GitHub

📂 **Project Structure**

src/main/java
│
|-- baseClass        ->Driver setup & initialization
|-- utils            -> Reusable methods & helpers
|-- ObjectRepository  -> POM Classes
|--
│
src/test/java
├── TestCases  -> All the testCases


---

✨ **Key Features**

* ✅ Page Object Model (POM) design pattern
* ✅ Reusable and maintainable test structure
* ✅ Cross-browser testing support
* ✅ Data-driven testing (basic)
* ✅ Maven-based project management
* ✅ Basic API testing integration

 ▶️ **How to Run the Project**

**🔹 Prerequisites**

* Java (JDK 8 or above)
* Maven installed
* Eclipse / IntelliJ IDE

 **🔹 Steps**

1. Clone the repository:
    git clone https://github.com/Rajnandini-DA/Selenium-Automation-Framework.git
   
2. Open project in IDE
3. Run tests using:

   ```
   mvn clean test
   ```

* Basic API request validation (Postman / RestAssured)
